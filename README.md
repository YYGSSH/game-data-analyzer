# JSON Data Visualizer

> ⚠️ **Work in Progress** – This dashboard is actively being developed. The feature set is evolving constantly, and the current implementation serves as a foundation for a **microservice-oriented data visualization tool**.

A **modern, interactive dashboard** built with **React**, **Tailwind CSS**, **Recharts**, and **Framer Motion** to visualize JSON datasets.
Users can filter data, search, and interactively select records to highlight metrics across charts and tables.

---

## Features

* ✅ **Interactive Table & Charts**: Click on a table row to highlight the corresponding metrics in charts and vice versa.
* ✅ **Filters & Search**: Filter by category, status, and search across department, supplier, or location.
* ✅ **Modern UI**: Clean, containerized design with Tailwind styling.
* ✅ **Responsive Layout**: Works on desktop and tablet layouts.
* ✅ **Animated Transitions**: Smooth entry and hover animations using Framer Motion.

> **Note:** Feature set is evolving. Additional chart types, export functionality, and API integration are planned as part of the microservice evolution.

---

## Project Vision

This project is designed to eventually serve as a **microservice** or component in a larger system, providing:

1. Interactive JSON data visualization.
2. Selectable, filterable datasets for downstream processing.
3. Extensibility for integration into a **data platform** or **BI microservice architecture**.
